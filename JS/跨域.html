导致跨域的原因是：浏览器同源策略的限制
            同源策略即：同一协议、同一域名、同一端口号

            解决跨域的常用方法有：
            new Image()、link、iframe
            
                JsonP跨域原理解析
            
            
                cors实现请求跨域
            
            框架提供的方式：
                devServer: {
                    proxy: {
                        "/api": {
                            target: "http://10.9.40.16:4000/",
                            changeOrigin: true,
                            pathRewrite: { "^/api": "" },
                            secure: false
                          }
                    }
                }
