javasript简称： JS
      JS分三个部分组成：
        1、ECMAscript标准--基础语法
        2、DOM document object model 文档对象模型
        3、BOM document object model 浏览器对象模型
      什么是JS ？
        ·是一门脚本语言
        ·是一门解释性语言
        ·是一门弱类型语言
        ·是一门基于对象的语言
        ·是一门动态类型的语言

      Javascript中的数据类型：
        javascript中有5中基本数据类型 （undefined，null，boolem，number，string）和1中复杂数据类型（object）
        基本数据类型：（值类型）
          Undefined、Null、Boolem、Number、String
        复杂数据类型：（引用类型）
          ·Object、
          ·Array、
          ·RegExp、
          ·Data、
          ·Function、
        基本包装类型：Boolem、Number、String
        单体内置对象：Global、Math

      类型检测：
        ·typeof
        ·instance of
        ·Object.prototype.toString.call()

      引用类型的地址在栈 对象在堆
      值类型和引用类型在内存中的存储方式：
        ·值类型按值存储
        ·引用类型按引用存储
      值类型复制和引用类型复制：
        ·值类型按值复制
        ·引用类型按引用复制
      值类型和引用类型参数传递：
        ·值类型按值传递
        ·引用类型按引用传递
      值类型和引用类型的差别：
        ·基本类型在内存中占据固定的空间，因此被保存在栈内存中
        ·从一个变量向另一个变量复制基本类型的值，复制的是值的副本
        ·引用类型的值是对象，被保存在堆内存中
        ·包含引用类型值的变量实际上包含的并不是对象本身，而是一个指向该对象的指针
        ·从一个变量向另一个变量复制引用类型值的时候，复制的是引用指针，因此两个变量最终都指向同一个对象


      javascript运行分为两个阶段
        ·预解析
          全局预解析（所有函数和变量声明都会提前，同名的函数和变量  函数的优先级高）
          函数内部预解析（所有的变量、函数和型参都会参与预解析）
            函数
            形参
            普通变量
        ·执行

    编程思想： 面向对象与面向过程
      1、面向过程就是亲力亲为，面面俱到、步步跟进、有条不紊
      2、面向对象就是找一个对象，指挥得结果
      3、面向对象将执行者转变成指挥者
      4、面向对象不是面向过程的替代，而是面向过程的封装

      面向过程：所有的事情都是亲力亲为，注重的是过程
      面向对象：提出需求、找对象、对象解决、注重的是结果 oop
      js不是一门面向对象的语言，是基于面向对象的语言，JS来模拟面向对象
      面向对象的特性：封装、继承、多态
      封装：就是包装，把一些重用的代码进行包装，在需求的时候直接使用
      继承：类与类之间的关系，JS中是没有类的概念的，js中有构造函数的概念，是可以实现继承的，是基于原型的
      多态：同一个行为，针对不同的对象，产生的不同效果


      对象：特定的某一个事物，一组无序属性的集合，具有属性和方法。
